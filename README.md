# BiNMF

Bilinear Nonnegative Matrix Factorization (BiNMF) implementation for
 hyperspectral unmixing.

## Files

-   BiNMF.py - Core algorithm
-   main.py - Example experiment
-   Synthetic_GBM.py - Synthetic GBM data generator

## Run

``` bash
python main.py
```

Generate synthetic data:

``` bash
python Synthetic_GBM.py
```

## Requirements

Python 3.x\
PyTorch\
NumPy\
SciPy\
Matplotlib
